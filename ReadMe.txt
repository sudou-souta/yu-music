友音楽とは
-------------
友音楽は、著作権が切れた曲を提供するサイトです。
音楽を提供したいという方はsudousouta9@gmail.comにmp3ファイルを添付してください。審査して(本当に著作権が切れているかなど)合格した場合は1日後以降に掲載します。
メールの形式：

Subject:「その音楽の名前」掲載について
To: sudousouta9@gmail.com

作曲者：作曲者の名前
曲名：曲名
リリースされた年(西暦)：〇〇年

その他必要だと思われる項目

添付ファイル(mp3)

添付ファイルについて
--------------
mp3ファイルを音楽ストリーミングサービスや動画共有サイトなどからとってくる場合は、その運営企業(または個人)から許可をとってください。

ご指摘について
-------------
・ご指摘は大変ありがたいものになります。(著作権が切れていない、曲名が違う、作曲者の名前が違うなど)
